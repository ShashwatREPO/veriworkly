# Creating Your First Resume

VeriWorkly provides an intuitive, high-performance editor designed to streamline the resume building process. This guide outlines the essential steps to create a professional document from inception to export.

## 1. Initializing the Document

From the main dashboard, select **Create New Resume**. You are presented with two options:

- **Blank Template**: Start with a clean slate to manually enter your data.
- **Import JSON**: Upload an existing VeriWorkly JSON schema to restore a previous document or migrate data.

## 2. Data Entry and Management

Utilize the sidebar navigation to populate the various sections of your resume. The editor provides structured fields for the following domains:

- **Personal Information**: Contact details, location, and professional social links (e.g., LinkedIn, GitHub).
- **Professional Experience**: Detailed work history. Utilize the integrated rich text editor to format bullet points and highlights.
- **Education**: Academic history, certifications, and relevant coursework.
- **Skills**: Granular skill management with support for custom categories.

## 3. Template Selection

The **Templates** tab allows you to switch visual layouts instantly. VeriWorkly ensures data integrity during transitions, allowing you to compare styles without data loss.

- **Modern**: A clean, sidebar-oriented layout optimized for readability.
- **Executive**: A formal, top-down structure designed for senior-level applications.
- **ATS Classic**: A minimalist, single-column layout optimized for maximum compatibility with Applicant Tracking Systems.

## 4. Visual Customization

The **Appearance** panel provides granular control over the document's aesthetic:

- **Color Palette**: Define the primary accent color to align with your personal brand.
- **Typography**: Select from curated font pairings optimized for print and screen.
- **Layout Adjustments**: Fine-tune margins, line heights, and section spacing to optimize page real estate.

## 5. Document Export

VeriWorkly utilizes a server-side rendering pipeline to ensure pixel-perfect exports.

- **PDF Export**: Generates a high-fidelity, ATS-optimized PDF document.
- **DOCX Export**: Provides an editable Word document for scenarios requiring further manual adjustments.

---

## Strategic Features

- **Section Visibility**: Toggle the visibility of individual sections using the "eye" icon. This allows you to maintain a comprehensive dataset while tailoring specific versions of your resume for different roles.
- **Section Reordering**: Drag and drop sections within the sidebar to modify the document's flow and hierarchy.
- **Master Profile Integration**: Utilize the [Master Profile](/docs/user-guides/account-management) to seed data across multiple documents, reducing redundant data entry.
