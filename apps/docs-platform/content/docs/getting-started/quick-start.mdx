# Quick Start

VeriWorkly is a full-stack platform comprising a **Next.js frontend** and an **Express.js backend**. The architecture is designed for high-fidelity document rendering and secure data synchronization.

## Architectural Components

The platform consists of several integrated components:

1. **Frontend (Resume Builder)**: A Next.js application handling user interaction and local-first state management.
2. **Backend API (Server)**: An Express-based service managing authentication, data persistence, and GitHub synchronization.
3. **Database (PostgreSQL)**: The primary relational store. We recommend [Neon](https://neon.tech) for managed deployments.
4. **Caching & Jobs (Redis)**: Facilitates rate limiting and manages the BullMQ job queue for document rendering.
5. **PDF Engine (Playwright)**: An asynchronous worker that renders resumes into ATS-optimized formats.
