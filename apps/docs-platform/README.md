# VeriWorkly Documentation Platform

The source code for [docs.veriworkly.com](https://docs.veriworkly.com). Built with Next.js and Fumadocs.

## 🚀 Quick Start

1. **Install dependencies** (from root):

   ```bash
   npm install
   ```

2. **Start development**:
   ```bash
   npm run dev:docs
   ```

The docs will be available at `http://localhost:3001`.

## ✍️ Writing Documentation

Documentation is stored as MDX files in `content/docs/`. The structure of the sidebar follows the folder structure of this directory.

## 📖 Live Docs

Visit [docs.veriworkly.com](https://docs.veriworkly.com) to view the rendered documentation.
