# VeriWorkly Blog Platform

The official blog for VeriWorkly, built with Next.js and Fumadocs.

## 🚀 Quick Start

1. **Install dependencies** (from root):

   ```bash
   npm install
   ```

2. **Start development**:
   ```bash
   npm run dev:blog
   ```

The blog will be available at `http://localhost:3002`.

## ✍️ Content Management

All blog posts are stored as MDX files in `content/blog/`. Simply add a new `.mdx` file to create a new post.
